# Clientes---Northwind---Proyecto-SQL
Análisis estratégico de clientes

Link artículo médium donde explico y desarrollo mejor: 
https://medium.com/@ezek41/an%C3%A1lisis-estrat%C3%A9gico-de-clientes-con-sql-northwind-92457ea4d39c

Motor de BD: SQL Server (SSMS).

Lenguaje: T-SQL.

Visualización: Google Sheets, Excel.


Ejecutar BBDD y luego consultas.

Consulta 1: EDA - ver rápidamente cantidad de clientes, ordenes, etc.

Consulta 2: Top 10 clientes por monto de facturación/compra

Consulta 3: monto de facturación por cliente (sin el filtro top 10)

Consulta 4: Modelo RFM (ultima compra, cuantas veces compran y monto)

Consulta 5: EDA y limpieza de datos para fechas de compra y fecha de envío

Consulta 6: Excluir ordenes sin fecha de envío (estas ordenes requerirán otro análisis que transciende al actual)

Consulta 7: Días transcurridos entre la compra y el envío por cliente (filtrando los top)

Consulta 8: Días transcurridos entre la compra y el envío por país
